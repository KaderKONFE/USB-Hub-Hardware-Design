# USB Hub Hardware & PCB Design

This repository contains the schematic and PCB design for a high-speed USB Hub, developed according to industrial design rules and signal integrity standards.

## Tech Stack & Tools
* **CAD Software:** KiCad
* **Design Type:** Schematic Capture & PCB Layout
* **Key Focus:** Signal integrity, power management, and protection circuits.

## Features
* Designed high-speed USB data lines with proper impedance matching.
* Integrated power management and circuit protection elements.
